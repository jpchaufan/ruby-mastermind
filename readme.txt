Classes:

	> MastermindGame
		Handles the overall game details

	> Game
		Handles each specific game that can be won or lost

	> Round
		Handles each round of guessing

	> ComputerAI
		Handles the computer's thinking

	> Helpers
		Helper functions

Game runs on the command line by running:
	ruby mastermind.rb

Game uses the ruby gem 'colorize'
